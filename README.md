# Shambel Mekuria Portfolio

This is my personal portfolio website built with modern web technologies.

## 🚀 Technologies Used

- Next.js
- Tailwind CSS
- Lucide React Icons
- Framer Motion

## 🌐 Live Demo

Visit the portfolio here:

[shambel-dev.vercel.app](https://shambel-dev.vercel.app/?utm_source=chatgpt.com)

## 📦 Getting Started

Clone the project:

```bash
git clone https://github.com/shambelmekuria/my_portfolio.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3000
```
